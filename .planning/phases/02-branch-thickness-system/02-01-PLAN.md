  <behavior>
    **buildCumulativeTimeMap(nodes, childrenMap):**
    - Takes an array of RevisionNode objects and a Map of nodeId -> children array (same structure already built inside TreeLayer's draw callback)
    - Returns Map<string, number> where each key is a node ID and value is that node's totalTime plus the sum of all descendants' totalTime
    - Leaf nodes: cumulative time = their own totalTime
    - Parent nodes: cumulative time = own totalTime + sum of all children's cumulative time (recursive with memoization)
    - Empty tree: returns empty map
    - Nodes with no totalTime (undefined/0): treated as 0 seconds
    - O(n) with memoization — each node computed exactly once

    Cases:
    - Single node with 300s totalTime -> Map { nodeId: 300 }
    - Parent(100s) -> Child(200s) -> Map { parent: 300, child: 200 }
    - Parent(0s) -> [ChildA(100s), ChildB(200s)] -> Map { parent: 300, childA: 100, childB: 200 }
    - Root(50s) -> Mid(30s) -> Leaf(20s) -> Map { root: 100, mid: 50, leaf: 20 }
    - Multiple roots: each subtree accumulates independently

    **Revised timeToThickness:**
    - Replace the existing timeToThickness (currently a multiplier from 1.0 to 3.5) with a direct pixel-width mapping
    - Input: cumulative seconds [0, 60, 300, 1800, 7200, 36000]
    - Output: pixel width [2.5, 4, 7, 13, 20, 28]
    - Logarithmic feel: rapid early growth, diminishing returns at high values
    - Clamps at boundaries (popmotion interpolate behavior): values below 0 return 2.5, above 36000 return 28
    - Export MIN_THICKNESS = 2.5 and MAX_THICKNESS = 28

    Cases:
    - timeToThickness(0) -> 2.5 (min, new branches visible)
    - timeToThickness(60) -> 4 (1 minute of study)
    - timeToThickness(300) -> 7 (5 minutes)
    - timeToThickness(1800) -> 13 (30 minutes)
    - timeToThickness(36000) -> 28 (10 hours, max)
    - timeToThickness(100000) -> 28 (clamped at max)

    **thicknessToColor:**
    - Maps thickness (pixels) to branch stroke color
    - Input: [2.5, 7, 13, 20, 28]
    - Output: ['#FDE68A', '#FCD34D', '#F59E0B', '#D97706', '#B45309'] (amber-200 through amber-700)
    - Returns rgba strings (popmotion interpolate output format)

    **thicknessToInactiveColor:**
    - Same input range as thicknessToColor
    - Output: desaturated slate colors with low opacity
    - Input: [2.5, 7, 13, 20, 28]
    - Output: ['rgba(148,163,184,0.3)', 'rgba(100,116,139,0.35)', 'rgba(71,85,105,0.4)', 'rgba(51,65,85,0.45)', 'rgba(30,41,59,0.5)']

  </behavior>
